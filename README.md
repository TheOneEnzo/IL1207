# Smart Helmet PCB

Battery-powered smart helmet system with:
- ESP32
- GPS (SIM7600E)
- MPU-6050 IMU

## Features
- Modular design
- 3.3V regulated supply
- Motion detection

###
This PCB is meant to fit inside a helmet and detect when the helmet is struck or fast acceleration changes. It should then send a emergency message via the SIM7600E to a reciever with the position of the accident.
